# DTN – Band Website

Official website for the band **DTN**.  
The site is built from scratch using **vanilla HTML, CSS, and JavaScript**, without frameworks, focusing on performance, clean visuals, and custom animations.

---

## Features

- Custom design (no W3CSS / Bootstrap)
- Scalable canvas logo (SVG to Canvas, retina-ready)
- Smooth scroll-based fade-in animations
- Spotify embed section
- Upcoming gigs / tour section
- Contact form with SweetAlert2 feedback
- Fully responsive (desktop and mobile)
- Lightweight and fast (no frameworks)

---

## Tech Stack

- HTML5
- CSS3  
  - Grid and Flexbox  
  - Custom animations  
  - Media queries
- JavaScript (Vanilla)  
  - Canvas API  
  - IntersectionObserver  
  - Scroll-based interactions
- SweetAlert2
- Font Awesome
- Spotify Embed
